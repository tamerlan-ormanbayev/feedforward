# CPSC-471 Data Base Management Systems Project
## Food Redistribution Marketplace
The application was built and tested using **Python 3.12**

**How to run:**
- pip install django
- cd feedforward
- python manage.py runserver

By default opens at http://127.0.0.1:8000/
